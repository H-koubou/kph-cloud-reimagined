# KPH Cloud Reimagined — リプレイス提案モック

環境プラットフォームクラウド（kph-cloud.jp）のリプレイス提案デモです。
静的HTMLのみ（Tailwind / GSAP / lucide はCDN）。

## 画面
| 画面 | ファイル |
|---|---|
| 提案トップ（ハブ） | `index.html` |
| 撮るだけ（写真AI） | `vision.html` |
| 会社の頭脳 | `mind.html` |
| しゃべるだけ事務 | `office.html` |
| 経営参謀 | `advisor.html` |
| 配車表 | `dispatch.html` |
| ドライバーアプリ | `driver.html` |
| マニフェスト | `manifest.html` |
| 経営ダッシュボード | `dashboard.html` |
| 提案書（PDF用） | `proposal.html` |

## ご注意
本デモ・資料は提案段階のものです。ご関係者内でのご確認をお願いし、第三者への再配布はお控えください。
